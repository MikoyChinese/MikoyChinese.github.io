2019 总结 | 展望 2020

Posted on 2020-01-18 | In Diary | Comments: | Views:

1. 2019年Flags:

  • 阅读20+本书籍，并为每一篇写读记。
  • 每天2篇英文文章和背诵15+单词。
  • 每周写一个小总结或周记，记录生活，记录点滴。
  • 搭建好个人博客，常常发布读记和总结和学习记录到博客。
  • 去云南或青海旅游，那些湖真的好美，希望多看看。
  • 理财收益10+%。
  • 再一次降低抽烟的数量，A packet 3 days.

2. 2019年完成度:

  • 阅读(20%):

    • Three days to see —— Hellen Keller
    • 毛泽东选集(第一卷)
    • Common Sense —— Thomas Paine
    • 感觉累了就冥想吧 —— 李上卿

  • 英文(57.534%)：

    • 扇贝阅读: 222天，2019年总365天。(60.822%)
    • 有道背单词: 198天, 2019年总365天。(54.246%)

  • 记录(13.207%)：

    • 2018, 再见 | 你好, 2019
    • Ubuntu Remote without installing desktop
    • 2019一季度总结 | 个人理财分析之保险
    • V-2-Ray Document(Server and Client)
    • Remote machine running ROS nodes
    • Install OpenCV 4 on Ubuntu 18.04
    • 2019三季度总结 | 5G时代的假想

  • 博客(100%): 已于一季度搭建完毕，并正式开放使用。Website: https://www.mikoychinese.top

  • 旅游(0%)： 年初虽然有出省几次，但不是旅游，也不是需求目的地。十一倒是体验了一下外岛，海水比海岸线要好很多。

  • 理财收益(100%+): 今年理财收益超额完成10%+, 虽然今年经历了康美事件，遭受大额亏损，但后期大市向好及选择的标的表现极佳，已收复康美的损失并超额完成既定目标。详细见三季度总结, 2019三季度总结 | 5G时代的假想。

  • 抽烟数量(100%): 由于受亿纬锂能影响，了解到了电子烟产品，已将部分场景传统香烟替换为电子烟，达到了A packet 3 days目标，后续希望可以全部场景替代并实现戒烟。

3. 2020年Flags:

  • 阅读10+本书籍，并为每一篇写读记。哈哈，看了一下2019年的完成量，看来得下放目标。
  • 每天2篇英文文章和背诵15+单词。
  • 每月写一个小总结或周记，记录生活，记录点滴。由周记改为月记。
  • 理财收益15+%。2020年可能成为新的投资年，随着楼市增长向缓，国内的资金有可能完成向资本市场转移，故调高预期，不过按今年1月至今收益，好像不小心已经完成了，那就继续保持吧。
  • 去云南或青海旅游，那些湖真的好美，希望多看看。上年没完成，希望今年有机会完成这个目标。
  • 再一次降低抽烟的数量，A package 5 days。继续加大电子烟场景覆盖，直至戒烟。
  • 实体经济投资。
  • 钢笔字练字，1月1书文。今年新到Pilot 78G+示范笔，槐荫书房先生打磨的金豪500, 还有坛水的鲁迅色墨水，可以好好尝尝鲜了。话说78g+铱粒也太小了吧，不好写，一点都不习惯，和Lamy 2000比差太远，期待先生打磨的笔的手感。
  • 运动，2019年运动量偏少，希望2020年，多跑跑步…

4. 2020年投资标的：

  原三季度内标的，部分已达到我的期望，故调整部分，并添加新的展望标的。Note: 原三季度文发表微信公众号2019-11-22日，但实际记录于2019年11月03日。

  • 原标的：(2019-11-03 -> 2019-11-22 -> 2020-01-18)

    • 三安光电：[15.38 -> 16.21 -> 21.04] : [0% -> 5.39% -> 36.80%]
    • 网宿科技：[9.56 -> 8.78 -> 9.85] : [0% -> -8.16% -> 3.14%]
    • 亿纬锂能：[36.71 -> 41.48 -> 54.88] : [0% -> 12.99% -> 49.49%]
    • 超图软件：[18.35 -> 18.90 -> 26.92] : [0% -> 2.99% -> 46.7%]
    • 天齐锂业：[23.01 -> 23.68 -> 32.23] : [0% -> 2.91% -> 40.06%]
    • 欧菲光：[11.92 -> 12.70 -> 16.93] : [0% -> 6.54% -> 42.03%]

  • 新标的：

    • 三安光电
    • 网宿科技
    • 亿纬锂能
    • 欧菲光
    • 嘉化能源
    • 东方能源
    • 万向钱潮

Note: 天齐锂业现阶段债务问题还比较严重，经历此次新能源浪潮，不符合我的理念了，故移除，如果能解决债务问题，会再考虑。超图软件本以为随着业绩慢慢上，而不是一突而就，但本身标的没问题，暂时放入观察区，不入仓。三安继续看好，网宿的话，预计年报不好看，只能等5G软件层及物联网时，才能爆发业绩，最重要的是老太婆手上的股份来个大佬收了它，那就一飞冲天。亿纬锂能，看好今年金豆电池和动力电池爆发，年收入100E，欧菲光，如果没有大额计提的话，今年业绩就会逐步回归，按往年业绩计算，也就25估值的科技股，等年报预告了，1月23日前必出，不出就GG。

  由于原标的都是科技股，虽然今年科技股可能会有较大表现，但为了分散风险，选择新加化工、资本、汽车零部件等标的。
