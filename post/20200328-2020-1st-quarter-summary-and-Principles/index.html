2020年一季度总结 | 原则

Posted on 2020-03-28 | Edited on 2020-04-11 | In Diary | Comments: | Views:

1. 前述：

  2020年是我国实现全面建成小康社会，实现第一个百年奋斗目标关键年，本来开端是大势向好，但年初受2019-nCoV疫情影响，对一季度经济造成了影响，希望后续随着国家政策利好在后续季度表现中弥补一季度中受影响的部分，完成今年目标。针对个人来讲，今年一季度算是经历了大起大落，后续回顾来看，一是没有明确自己的原则，二是没有把握好自己的原则，以至于得失参半，但赛翁失马，焉知非福，日子还在继续，失去一部分，能换回教训及经验也是值得的。

2. 2020年一季度总结：

  • 阅读方面:

  本季度阅读了《毛泽东选集第二卷》，Jesse Livermore’s《Reminiscences of a stock operator》，Ray Dalio’s《Principles》，阅读方面本季度尚可，说来也是巧合，几本书中也阐述了大起大落的过程，毛泽东选集第二卷是讲述1937年至1941年时期，也是中华民族进行全面抗战的起点。《Reminiscences of a stock operator》讲述Jesse Livermore人生中的大起大落，并警示后人。《Principles》是对Ray Dalio过往经验的总结。其中较深刻的是，他们在失败后的不气馁，自信，并对过程进行分析总结，并根据实际反馈一步一步验证，调整，最终成为了大多数人渴望的大多数。读万卷书不如行万里路，行万里路不如阅人无数；阅人无数不如名师指路，名师指路不如自己去悟。不亲身经历，你又如何读懂简单的字里行间映射出的人生百态？

  • 英文学习:

  扇贝打卡共37天，有道背单词共30天，从开年到今天3月28号共88天。回顾整个季度，二月份惨败，学习自主性大幅下跌，导致学习完成率仅42.04%，34.09%。由于《Principles》阅读的是中文版，怕其中翻译或夹带个人理解，这段时间通过电子英文版的《Principles》再重读一遍，也当作对英文学习的弥补。

  • 月记方面:

  月记方面比较杂，一月份完成Hellen Keller’s《Three Days To See》阅记，二月份写了技术方面的《基于SSIM提取单一背景下目标Mask及Edge》，三月份今天是写了这篇一季度总结，明天还会写一份关于LSP 3D keypoints To SMPL Model的技术分享。文章均记录在博客及个人公众号内。

  • 理财方面:

  一季度的总结两个字，白干。回顾得失，是自己被巨大的收益蒙蔽了初心，放弃了原则，从而导致本高额的收益又失去。让我不得不想起Principles内的一句话：”You can’t never be sure of everything: There are always risks out there that can hurt you badly, even in the seemingly safest bets, so it’s always best to missing something.” 1月份本已完成今年年度目标，二月份欧菲光也达成我既定的翻倍目标，收益也超额完成自己预期，但出于贪婪及对自己巨大获益的乐观，导致误判了外围疫情导致的黑天鹅事件，截至本周，将今年收益全部回吐，正式白干。望自己不忘初心，坚定原则！

  • 抽烟方面: 电子烟的场景覆盖没能实现，现阶段依旧维持在1 package 2~3 days.
  • 实体经济投资: 目前在着手准备着。
  • 练字及运动: 本季度练字断断续续，静不下心，看来我又得好好看看我那本冥想的书了，心不静写出来的字也是差强人意，就不展示了。运动方面，受疫情影响，完成率为0%，哈哈，拿疫情当幌子，就是懒。

3. 2020年投资标的：

  出于年初制定的抵抗风险原则，于20200118将标的分为了科技和抵御风险板块，但经过验证，发生无法预知的风险时，整体趋势都是下滑的。故而会调整部分标的，由于对经济无法预估，二季度不提供任何建议，新标的是对下半年的一个参考。下为记录标的一季度表现，分别为初始价，现价，及期间最大值。

    • 三安光电: [21.04 -> 20.09 -> 31.63] : [0% -> -4.51% -> 50.33%]

    • 亿纬锂能: [54.88 -> 55.12 -> 85.26] : [0% -> 0.44% -> 55.36%]
    • 欧菲光: [16.93 -> 14.41 -> 23.23] : [0% -> -14.88% -> 37.21%]
