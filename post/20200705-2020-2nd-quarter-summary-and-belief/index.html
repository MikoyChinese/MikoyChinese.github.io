2020年半年度总结 | 信仰

Posted on 2020-07-05 | Edited on 2020-07-11 | In Diary | Comments: | Views:

1. 前述：

  时间来到了2020的七月份，回顾半年，却总觉得没有真正得到什么，或者能深刻回忆的事情，时间过却了，却如白驹过隙，好像看到了身影，但仍是一片白茫。幸得有立flags，至少也能捕捉一些时间过去时留下的影子。前段时间，看了《局中人》中沈父与周达元探讨信仰的问题，感觉还是很困惑，故而也会探讨一些信仰的问题。

2. 2020年一季度总结：

  • 阅读方面:

  本季度暂未开展阅读计划，仅补阅了上季度未完成的电子英文版的《Principles》。为了完成年度目标，预计下季度要读3-4本了，加油！！！

  • 英文学习:

  扇贝打卡共55天，有道背单词共30天，二季度共91天，由于单词词本背诵完毕，故而有道背单词6月整个月未背诵。英文阅读及背单词完成率分别为60.44%，49.18%(仅4、5月)。预计下季度会从新找个词本背诵。

  • 月记方面:

  由于本季度未进行阅读，未记录月记及读记。

  • 理财方面:

  二季度是大翻身行情，但又让人看不懂，由于四月是煎熬行情，忍不住诱惑看了看路边的野花，被套20个点，故而没能买到二季度比较看好的标的，错失了上升行情。好在后续回升，解套出局，回归确立的标的，逐步实现收益。下半年是明确看好汽车智能化及mini Led的，但由于特斯拉回归太猛，导致观察标的中的汽车智能化几只大幅度上浮，失去了投资价值。但还是看好，等待他们回调，然后再入手，故而下季度投资标的会有很大调整，详见文末。

  • 抽烟方面: 现阶段依旧维持在1 package 2~3 days.
  • 实体经济投资: 目前在着手准备着。
  • 练字及运动: 本季度完成度为0%。

3. 2020年投资标的：

  一季度制定的标的中，部分进行了分红及送股，所以股价需要除权处理，部分标的受非理智因素影响，产生了巨大收益，就当市场给的红包吧，谁拿住了，我在此恭喜你，哈哈。下为记录标的二季度表现，分别为初始价，现价，及期间最大值。Note: 初始价格变动是由于除权导致。

  • 原标的：(2020-03-28-> 2020-07-05 -> Max)

    • 三安光电: [20.80 -> 25.28 -> 27.10] : [0% -> 21.54% -> 30.29%]

    • 亿纬锂能: [28.91 -> 47.12 -> 49.92] : [0% -> 62.99% -> 72.67%]
    • 欧菲光: [14.41 -> 19.24 -> 19.45] : [0% -> 33.52% -> 34.98%]
    • 东方能源: [4.46 -> 4.55 -> 4.70] : [0% -> 2.02% -> 5.38%]

    • 德赛西威: [31.37 -> 61.80 -> 64.77] : [0% -> 97.00% -> 106.47%]

  欧菲光，老朋友了，大家等待它的半年报，如果单独二季度利润大于5亿，那就坚定拿好，很快就会与20以下告别了。东华软件，由于在增发阶段，现在是很好的入手机会，增发落地，15起步，是可以中长线投资的选择。德赛西威，这段时间上的太猛，都已经翻倍了，故而等待回调，不建议此时入，投资是基于业绩的，业绩不达标，不可急于下手，但它未来发展很让人值得期待，是汽车智能化中的绝佳标的。

  亿纬锂能随着思摩尔香港上市，还会有一次炒作行情，估计会达到千亿市值，从19年看到现在，经历2年8倍，可惜，赚了20个点就走了。本次移除，不代表不看好，只是有更好的选择。赚自己能力范围内的钱，才拿的稳！

Warning: 理财有风险，投资需谨慎！！！

4. 信仰:

电影中原对话：

这个世界是需要改变的，我您肯多受些痛苦，也想看到这个世界，有所改变。

改变是需要代价的，可是事情都是需要代价的。

你不怕死？

怕， 但如果我的死能给这个世界带来改变，哪怕是一点点，值了。

  我也一直在寻找我的信仰，佛算一个，是我在迷惘无知的时候可以选择虔诚的对象。但我追求的信仰我暂未找到，前辈们的共产主义信仰确实让我折服，什么时候我能找到让我追求一生的信仰呢？而真正的信仰又是什么呢？如果现在不明白，就让时间和经历帮忙寻找答案吧。保持阅读，保持行走，保持思考。

  看多了大多数，看了《沉默的大多数》及《乌合之众》就越怕自己丧失思考，丧失追求，丧失信仰。群众的力量是可怕的，群众的力量是可指导的。马斯洛提出了需求层次理论：生理需求，安全需求，社交需求，尊重需求，自我实现。而群众会止步于第三个需求，然后被外界短期刺激快感麻痹，导致回归到追求第一和第二需求，周而复始，止步不前。希望自己可以跳脱此莫比乌斯环！
